# Empirically-grounded-foodweb-model
The repository is organized into Code, Figures and Data. The Code folder consists of the python script for generating food web model, Foodweb_model.py, the script for perturbing parameters in the food web model, Sensitivity_analysis.py and the script for multiple simulations of the food web, Correlation.py. The data generated from the Sensitivity_analysis.py and Correlation.py can be found in the Data folder as parameter_sensitivity.xlsx and community_outcomes.xlsx. Figures consist of the code used to generate the final figures from the generated data.
